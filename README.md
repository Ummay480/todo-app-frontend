# Todo App Frontend

A modern, responsive web frontend for the Todo App built with Next.js.

## Tech Stack
- **Framework**: Next.js 16+ (App Router)
- **Styling**: Tailwind CSS
- **Authentication**: JWT-based with localStorage
- **Icons**: Lucide React
- **API**: Integrated with Todo App backend API

## Features
- User authentication (login/signup)
- Task management (create, read, update, delete)
- Task prioritization (High, Medium, Low)
- Task completion toggling
- Search, filter, and sort capabilities
- Responsive UI with Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

## Environment Variables
- `NEXT_PUBLIC_API_URL`: URL of the deployed backend (e.g., `https://todo-app-backend-production-2bc8.up.railway.app`)

## Deployment
This project is configured for deployment to Vercel with API rewrites to connect to the backend service.
